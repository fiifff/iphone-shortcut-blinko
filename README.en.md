## iPhone Shortcut for Blinko

* A lightweight iOS Shortcut that lets you quickly send content to Blinko and create a new Flash Note with minimal friction.

* Shortcut Link:
  https://www.icloud.com/shortcuts/09583ece893847659a174a4ef53c7051

## Features

* Direct file and text sharing: Instantly send files or text to Blinko to create a new Flash Note, without any tag‑selection step.
* Smart handling for text & Safari content: These two types are not uploaded as attachments. Instead, the original text or URL is appended directly after your input text.

## How to Use

* Enter your low‑permission access token
* Set your base URL — for example: https://example.com
