# iphone-shortcut-blinko
Apple iphone IOS 捷径--blinko(Apple Iphone IOS shortcut for blinko)

## blinko捷径

### 地址

* https://www.icloud.com/shortcuts/09583ece893847659a174a4ef53c7051

### 功能

 * 直接分享文件和文本到blinko，直接创建一个新的闪念，不提供标签设置。
 * 对文本和Safari做了排除，这两种类型不会以附件上传到blinko，而是以原本的文字或者链接追加在输入的文本之后。
   

### 使用方法

1. 填写低权限访问令牌
2. 基本的url。例如：https://example.com
